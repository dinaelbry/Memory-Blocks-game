**Memory Game using Vanilla JavaScript.**

This project is a card matching game built from scratch to focus on logic, state control, and user experience.
No frameworks. No shortcuts.

**Features**
Card shuffle logic
Clear game states idle, playing, paused, finished
Accurate timer with pause and resume
Score system based on time and wrong attempts
Leaderboard saved in localStorage
Last player and top players ranking
Responsive layout for desktop and mobile
Dark mode
